About This Movie App

Welcome to the Movie App — a simple, clean, modern web application designed to help you explore movies currently playing in theaters. Browse trending titles, view detailed information, and save your favorites for later.

How It Works

  • TMDB API provides the list of movies currently playing.
  • OMDb API supplies detailed movie information.
  • YouTube API fetches official trailers.
  • Your favorite movies are saved locally in an internal database.

Features

  • Clean, modern UI with responsive layout.
  • Movie details: Posters, cast, plot, ratings, and more.
  • Trailer viewing directly on the movie page.
  • Ability to add/remove movies from favorites.

Credits

This app uses public APIs including TMDB, OMDb, and YouTube Data API.

Built with ❤️ using Python, Flask, HTML, and CSS.
